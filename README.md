\# 🎮 Guess Game Pro



A fun command-line number guessing game built with Node.js.



\## 🚀 Features



\- 🎯 Choose difficulty (Easy / Medium / Hard)

\- 🔥 Smart hints (Very close / Close / Far)

\- 🔄 Play again option

\- 📊 Tracks number of attempts

\- 🎮 Interactive terminal game



\## 🧠 How to Play



1\. Run the game using Node.js

2\. Choose a difficulty level

3\. Try to guess the secret number

4\. Get hints after each guess

5\. Win in as few attempts as possible!



\## ▶️ Run the Project



```bash

node game.js
