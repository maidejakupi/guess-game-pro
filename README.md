# 🎮 NumQuest — Guess the Number

> A sleek browser-based number guessing game with difficulty modes, proximity hints, and a best-score tracker.

![HTML](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=black)

---

## 🚀 Features

- 🎯 Three difficulty modes — Easy (1–100), Medium (1–500), Hard (1–1000)
- 🔥 Live proximity hints — scorching hot, warm, cold, and ice cold
- 📊 Attempt counter with personal best score saved to localStorage
- 🏆 Share your result as an emoji grid
- 💅 Modern dark UI with animated background and glassmorphism card
- 📱 Fully responsive — works on mobile and desktop

---

## 🧠 How to Play

1. Open the game in your browser
2. Choose a difficulty level from the settings (⚙)
3. Type your guess and hit **Guess** (or press Enter)
4. Read the hint — the card border glows hotter as you get closer
5. Win in as few attempts as possible and beat your best score!
